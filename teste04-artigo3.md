# **Multimodal Learning Analytics**

| **Conceito** | **explicação** |
| --------- | -------------- |
| **Multimodal Learning Analytics** | **Multimodal Learning Analytics (MMLA) é um campo de pesquisa que usa uma variedade de fontes de dados para obter insights sobre como os alunos aprendem. Esses dados podem incluir vídeo, áudio, logs, gestos e biossensores. Técnicas de MMLA podem ser usadas para rastrear o progresso dos alunos ao longo do tempo, identificar áreas onde eles estão com dificuldades e fornecer feedback para ajudá-los a melhorar seu aprendizado.** |
